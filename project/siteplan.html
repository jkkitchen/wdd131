Final Project

Site Name

Quilting in Greenville

This website will focus on quilting resources in Greenville, SC.

Site Purpose

The purpose of this website will be to point local quilters to nearby resources (such as quilt shops, guilds, and other places to find fabric), online tutorials, and fun pattern and fabric designers.

Scenarios

Where can I go to find Moda fabrics?

Where can I find longarm quilting services?

What classes area available in my area?

Color Schema

Main Color: Lapis Lazuli #33658A

The main color will be used for the top header and as background in strategic places. It will also be used as a font color on a white background for other headers.

Accent Colors: Carolina Blue #86BBD8, Moss Green #758E4F, Chili Red #D64933, and Gray #434242

The accent colors will be used for the navigation bar, borders, and buttons.

Font Colors: White, Blue, and Gray

The main blue color, white, and a dark gray will be used as the font colors on this webpage.

Typography

Fonts: Winky Sans for headings, Inter for text
